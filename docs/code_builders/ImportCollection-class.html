ImportCollection

ImportCollection class

A collection of import directives for batch creation.

This class provides convenience constructors for creating multiple import statements at once, reducing boilerplate code when importing several files of the same type.

Example:

var imports = ImportCollection.package('flutter', [
  'material.dart',
  'widgets.dart',
  'gestures.dart',
]);
