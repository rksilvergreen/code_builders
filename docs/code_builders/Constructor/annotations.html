annotations

annotations property

List<String> annotations
final

Optional list of annotations for this constructor.

Annotations are metadata that appear before the constructor declaration. Each string should be the annotation without the '@' symbol.

Common annotations:

  • 'deprecated' - Marks the constructor as deprecated
  • 'override' - Not typically used for constructors
  • Custom annotations like 'JsonConstructor()'

Example:

Constructor(
  className: 'User',
  annotations: ['deprecated', 'JsonConstructor()'],
)

Generates:

@deprecated
@JsonConstructor()
User(...);
